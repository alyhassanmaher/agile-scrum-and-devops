# ShopEase — Agile, Scrum & DevOps Project

A college project demonstrating Agile methodologies, Scrum framework, and DevOps practices through the planning and design of **ShopEase**, an e-commerce platform concept.

## Contents

| File | Description |
|------|-------------|
| `Agile, Scrum and DevOps.pdf` | Full project report covering Agile principles, Scrum ceremonies, and DevOps pipeline design |
| `ShopEasePresentation.pptx` | Presentation slides for the ShopEase project |

## Topics Covered

- **Agile Manifesto** — principles and values
- **Scrum Framework** — roles (Product Owner, Scrum Master, Dev Team), sprints, backlog grooming, retrospectives
- **DevOps Practices** — CI/CD pipeline, automation, collaboration between Dev and Ops
- **Product Design** — ShopEase e-commerce platform requirements and planning
